线程安全的对象生命期管理

线程安全的定义

满足三个条件：
１)多个线程同时访问时，表现出正确的行为
２)无论操作系统如何调度线程，无论这些线程的执行顺序如何交织
３)调用端代码无须额外的同步或其他协调动作

对象的创建要求

１)不要在构造函数中注册任何回调
２)不要在构造函数中把this指针转给跨线程的对象

因为在构造函数执行期间对象还没有完成初始化，如果this被泄露给其他对象，其他线程可能访问这个半成品对象造成问题
